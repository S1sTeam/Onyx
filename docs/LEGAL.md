# Licensing Notes (Native Mode)

Current native runtime artifacts are built from source in this repository:
- Onyx launcher (`src/`)
- OnyxProxy native (`native/onyxproxy`)
- OnyxServer native (`native/onyxserver`)

## Distribution checklist

1. Include `dist/licenses/onyx-native-NOTICE.txt`.
2. Keep `dist/versions.txt` for source/build traceability.
3. If you distribute modified binaries, keep corresponding source available for your users.

## Upstream reference folders

`upstream/` may remain in the repository for reference/history.
Native build pipeline does not require upstream artifacts.
